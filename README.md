- Javascript
- Express

- Arrow Functions
- Automated tests

# Libraries and Technologies
- NodeJS
- Express
- EJS
- Axios
- CI/CD
- Jest
- Vercel

# How to run locally
- Clone the repository
- Run NPM install
- Run NPM start
- App will be running on localhost:3000
